SVG 简介

概念

  • SVG 指可伸缩矢量图形 (Scalable Vector Graphics)
  • SVG 用来定义用于网络的基于矢量的图形
  • SVG 使用 XML 格式定义图形
  • SVG 图像在放大或改变尺寸的情况下其图形质量不会有所损失
  • SVG 是万维网联盟的标准
  • SVG 与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体

特点

  • SVG 易读取或修改
  • SVG 与 JPEG 和 GIF 相比，尺寸更小，可压缩性更强
  • SVG 是矢量的，可被高质量打印
  • SVG 图像中的文本是可选的，同时也是可搜索的（很适合制作地图）

关注点

  • 矢量
  • 文件小
  • 多色支持，可以随时变色
  • 动画可与CSS、JS配合
  • 兼容IE9

行业用法

  • 矢量图
    • 背景图或者图标，如苹果官网logo、Antd Icon
  • SVG 动画，更多可能性
    • 纯CSS动画
    • SVG动画
  • SVG Sprite
    • symbol 定义形状，以id命名并使用，
    • CC这边业务相对分散，图标尚未集中管理

  • 矩形 <rect>

  • 椭圆 <ellipse>

  • 折线 <polyline>
  • 多边形 <polygon>

SVG 动画兼容性

化简为繁 - 图形变换即可

// 图形变换
transform: rotate(-45deg) skew(15deg, 15deg) translate(-21%, -27%) scale(1.1);
// 路径的转角处使用的形状或者绘制的基础形状 为 圆形

// 增加描边，来突出转角处的圆角
stroke-width: 30;
